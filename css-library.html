API Reference

Take a demo

Class Description Example
.optimus General attributes for FORM 
<form class="optimus"> </form>
.grid-$-12 How many ($) of 12 equal columns the container fills 
<div class="grid-12-12">. . .</div>
.optimus-req Attributes to customize REQUIRED asterisk 
<em class="optimus-req">*</em>
.optimus-list Attributes to customize LIST of INPUT RADIO or CHECKBOX 
<ul class="optimus-list">   <li>    <input name="" type="radio" />    <label for="">Opt. 01</label>   </li>   <li>    <input name="" type="radio" />    <label for="">Opt. 02</label>   </li></ul>
<ul class="optimus-list">   <li>    <input name="" type="checkbox" />    <label for="">Opt. 01</label>   </li>   <li>    <input name="" type="checkbox" />    <label for="">Opt. 02</label>   </li></ul>
.optimus-button Attributes to customize an INPUT BUTTON 
<input class="optimus-button" type="submit" value="send" />
Floating
.clear Specify which column where other floating elements are not allowed 
<div class="grid-3-12 clear">   <input type="text" value="Fill up the field" /></div>
.right Attributes for float elements to the right 
<input class="optimus-button right" type="submit" value="send" />
.left Attributes for float elements to the left 
<input class="optimus-button left" type="submit" value="send" />
.optimus-equal Attributes for equal height elements (fixing float bugs) 
<div class="grid-3-12 optimus-equal">   <input type="text" value="Fill up the field" /></div>
Size
.optimus-small Define the width of form elements as SMALL 
<input type="text" class="optimus-small" value="Fill up the field" />
.optimus-medium Define the width of form elements as MEDIUM 
<input type="text" class="optimus-medium" value="Fill up the field" />
Message box
.optimus-msg-info Custom message for information 
<div class="optimus-msg-info"><h3>Information Message</h3></div>
.optimus-msg-warning Custom message for warning 
<div class="optimus-msg-warning"><h3>Warning Message</h3></div>
.optimus-msg-error Custom message for error 
<div class="optimus-msg-error"><h3>Error Message</h3></div>
.optimus-msg-success Custom message for success 
<div class="optimus-msg-success"><h3>Success Message</h3></div>
